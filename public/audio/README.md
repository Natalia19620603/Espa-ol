# Аудио файлы для уроков

Эта папка содержит аудио файлы для текстов чтения и упражнений на произношение.

## Текущий статус
Сейчас здесь находятся файлы-заглушки.

## Что нужно сделать
Для полной функциональности необходимо:
1. Записать настоящие аудио файлы с испанским произношением текстов
2. Заменить файлы-заглушки настоящими MP3 файлами
3. Использовать сервисы TTS (Text-to-Speech) или записать с носителями языка

## Формат файлов

### Аудио для текстов чтения
- lesson-[модуль]-[урок]-reading.mp3
- Например: lesson-2-2-reading.mp3 для модуля 2, урока 2

### Аудио для упражнений на произношение
- **consonantes.mp3** - общее аудио с произношением испанских согласных звуков (используется в упражнениях 7-12 модуля 1.1)
- Разместите файл consonantes.mp3 в этой папке (/public/audio/)

### Аудио для Модуля 2 Урока 3
- **Модуль 2 Урок 3.mp3** - основное аудио для урока
- **Модуль 2 Урок 3 - Аудио 2.mp3** - дополнительное аудио
