# Аудио файлы для уроков

Эта папка содержит аудио файлы для текстов чтения в уроках.

## Текущий статус
Сейчас здесь находятся файлы-заглушки.

## Что нужно сделать
Для полной функциональности необходимо:
1. Записать настоящие аудио файлы с испанским произношением текстов
2. Заменить файлы-заглушки настоящими MP3 файлами
3. Использовать сервисы TTS (Text-to-Speech) или записать с носителями языка

## Формат файлов
- lesson-[модуль]-[урок]-reading.mp3
- Например: lesson-2-2-reading.mp3 для модуля 2, урока 2
